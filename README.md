# SkillSwap — Платформа для обмена навыками

## О проекте

**SkillSwap** — это полнофункциональное веб-приложение для обмена знаниями и навыками между пользователями. Платформа позволяет людям публиковать свои компетенции и находить партнёров для взаимного обучения.

Пользователи могут:
- Создавать профиль и публиковать навыки, которыми готовы делиться ("Учу")
- Указывать навыки, которые хотят получить ("Учусь")
- Просматривать профили других пользователей отсортированные по популярности и новизне
- Добавлять интересные профили в избранное (для авторизованных пользователей)
- Управлять своей деятельностью и активностью

## Ключевые особенности разработки

### Архитектура и управление состоянием
- **Redux Toolkit** с асинхронными операциями (thunks) для управления глобальным состоянием
- **Private Routes** для защиты страниц, требующих авторизации
- Восстановление состояния пользователя из `sessionStorage` при перезагрузке приложения

### Компоненты и структура
- Модульная архитектура с четким разделением на `features`, `pages`, `widgets` и `shared` компоненты
- **CSS Modules** для изолированных стилей компонентов и предотвращения конфликтов имен классов
- Переиспользуемые UI компоненты (Button, Tag, Dropdown, UserCard и др.)

### API интеграция
- Полная интеграция с RESTful API для получения, обновления и управления данными пользователей
- Обработка ошибок на уровне API слоя с логированием
- Поддержка различных типов запросов (GET, PUT) для работы с карточками пользователей

### Тестирование и качество кода
- **Unit тесты** на Jest для Redux слайсов и критичной бизнес-логики
- **Storybook** для документирования и визуального тестирования компонентов
- **ESLint** и **Prettier** для поддержания единого стиля кода

## Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Шаги установки

1. **Клонируйте репозиторий**

2. **Установите зависимости**
```bash
npm install
# или
yarn install
```

3. **Настройте переменные окружения**

Создайте файл `.env` в корне проекта:
```env
```VITE_API_URL = 'https://68836e0321fa24876a9e147f.mockapi.io/api/sillswap'```
```

4. **Запустите проект на локальной машине**
```bash
npm run dev
# или
yarn dev
```

Приложение откроется по адресу `http://localhost:5173`

### Доступные скрипты

```bash
# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Проверка кода линтером
npm run lint

# Форматирование кода Prettier
npm run prettier

# Запуск unit тестов
npm test

# Просмотр Storybook компонентов
npm run storybook

# Сборка Storybook
npm run build-storybook
```

## Структура проекта

```
src/
├── api/              # API слой для работы с серверными данными
├── app/              # Главный компонент приложения и маршруты
├── features/         # Функциональные блоки (UserCards, Skills, Favorites)
├── pages/            # Страницы приложения
├── shared/           # Переиспользуемые компоненты, утилиты, хуки
├── slice/            # Redux слайсы для управления состоянием
├── widgets/          # Сложные компоненты (Login widget, UserCard widget)
└── stories/          # Storybook истории компонентов
```
